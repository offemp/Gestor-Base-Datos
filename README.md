A basic desktop application built with Python for managing a residence database using **SQLite** for data persistence and **Tkinter** for the graphical user interface (GUI).

## Features

This application allows for comprehensive resident management, including:

* **Export to Excel**: Save the current resident data to an Excel file.
* **Import from Excel**: Load resident data from an existing Excel file into the database.
* **CRUD Operations**:
    * **Create**: Add new residents.
    * **Read (Retrieve)**: View and search existing residents.
    * **Update**: Edit a resident's information.
    * **Delete**: Remove a resident from the database.
